Para subir a API e o front, siga os passos abaixo:
1- Faça um git clone do repositório
2- no terminal navegue até o diretorio api-rest e insira o comando de inicialização npm run dev (a aplicação vai subir na porta 1234)
3- abra outro terminal, navegue ate o diretório vehicles-front e insira o comando de inicialização que também é npm run dev (a aplicação vai subir na porta 3000)
4- abra o navegador e acesse a url: http://localhost:3000 para acessar a aplicação (por aqui é possível acessar o campos: ver todos os carros e cadastrar carros)
5- para acessar os demais comandos do back-end, favor abrir o ThunderClient na url: http://localhost:1234.
